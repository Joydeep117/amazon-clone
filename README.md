
# Amazon Clone

A responsive HTML and CSS clone of Amazon's homepage interface. This project demonstrates modern web design principles with a clean, user-friendly layout that mimics Amazon's visual style.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface inspired by Amazon
- **Header Navigation**: Includes logo, search bar, account links, and shopping cart
- **Product Categories**: Grid layout showcasing different product categories
- **Hero Banner**: Eye-catching banner section with call-to-action
- **Footer**: Comprehensive footer with links and legal information

## Files Structure

```
amazon clone/
├── index.html      # Main HTML file
├── styles.css      # CSS stylesheet
└── README.md       # This file
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required - it's pure HTML and CSS!

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox)
- Responsive Design Principles

## Customization

You can easily customize this project by:

- Modifying colors in the CSS variables section of `styles.css`
- Replacing placeholder images with your own
- Adding more product cards or sections
- Updating links and content in `index.html`

## Notes
